# Tauri Mobile Migration Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Migrate Orion Chat from a desktop-only Tauri app to a production-ready Tauri Mobile app that runs on iOS and Android with a usable chat flow, mobile-safe settings, persistent storage, and a mobile-first layout.

**Architecture:** Keep the current `SvelteKit + Tauri v2 + Rust` architecture and introduce a thin platform abstraction layer instead of rewriting business logic. Split desktop-only capabilities from cross-platform features, move path handling onto Tauri app paths, then adapt the frontend layout and interactions for touch devices before adding any native mobile plugin for export/share gaps.

**Tech Stack:** Tauri v2 Mobile, Rust, Svelte 5, SvelteKit, TypeScript, Tauri capabilities, Xcode, Android Studio.

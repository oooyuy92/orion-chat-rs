# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-03-08

First public release of Orion Chat.

### Added
- Desktop AI chat client built with Tauri v2, SvelteKit, and Rust
- Multi-provider support for OpenAI-compatible APIs, Anthropic, Gemini, and Ollama
- Local conversation storage with SQLite and full-text search
- Streaming responses, message regeneration, and message version switching
- Assistant presets and per-model parameter configuration
- Conversation auto-rename and auto-compress workflows
- Markdown/JSON conversation export, local backup, cache cleanup, and app data reset
- Responsive desktop UI built with Svelte 5 and shadcn-svelte style components
